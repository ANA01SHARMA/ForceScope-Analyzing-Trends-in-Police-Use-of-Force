# ForceScope: Analyzing Trends in Police Use of Force (2016–2022)

This project is an interactive Excel-based dashboard designed to explore 8203+ recorded Police Use of Force incidents between 2016 and 2022.  
The dashboard provides a clean, visual, and filterable interface to analyze trends across neighborhoods, demographics, force types, and timelines.

## Dashboard Overview

The dashboard includes:
Dynamic slicers for filtering by:
  - Subject Armed Status  
  - Sex  
  - Race  
  - District  
  - Neighborhood  
  - Force Description  
  - Year of Incident  

  Interactive charts such as:
  - Time-series trend of incidents  
  - Neighborhood-wise incident distribution  
  - Race-wise breakdown  
  - Annual incident comparison  
  - Pie chart showing proportion of force types  
  - Weekly/Monthly spike analysis  

  Key Metrics Displayed
  - Total Incidents: 8203  
  - Most Affected Age Group: 25–34 years  

## Project Structure

# Objectives of the Analysis
- Identify trends in use-of-force incidents over the years  
- Compare incident rates across districts and neighborhoods  
- Analyze demographic patterns (age group, sex, race)  
- Detect spikes or anomalies in force usage  
- Evaluate the most common types of force used  

# Tools & Techniques Used
| Component | Technology Used |
|----------|----------------|
| Dashboard | Microsoft Excel |
| Charts | Pivot Charts, Slicers, Timelines |
| Data Cleaning | Excel Power Query |
| Data Handling | Pivot Tables, Functions, Filters |
| Visualization | Line charts, Bar charts, Pie charts |

# How to Use
1. Download the Excel file from the dashboard folder  
2. Enable editing + ensure macros/slicers are active  
3. Use slicers on the left and right panels to interact with the visuals  
4. Explore insights by applying multiple filters  

# Key Insights from the Dashboard
- 25–34 years is the most affected age group
- Several districts show **significantly higher incident counts** (e.g., Bayview, Mission, etc.)  
- Firearm-related force incidents show periodic spikes  
- A noticeable variation occurs between years 2016–2022, with fluctuation in volume
